체크:
1. **특금법 준수**: ISMS+실명계좌 거래소만 (업비트/빗썸/코인원/코빗)
2. **세금 준비**: 거래 기록 시간/가격/수량/수수료 영구 저장
3. **자금세탁 방지**: 타인 자금 운용 기능 없음 (개인 전용)
4. **공무원 겸직**: 구독/광고/수수료 수익 기능 없음
5. **개인정보**: 다른 사용자 데이터 저장 없음 (단일 사용자 앱)

위반 시 구체적 파일/기능 지적 및 제거 권고.
